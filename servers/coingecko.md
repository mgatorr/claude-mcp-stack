# coingecko

**Category:** crypto · **Runtime:** npx · **Package:** `@coingecko/coingecko-mcp`

Cryptocurrency market data — prices, market caps, and coin metadata — via
[CoinGecko](https://www.coingecko.com).

## API key

Not required for basic usage; the public CoinGecko API is used by default. (CoinGecko offers
paid API keys for higher rate limits, but this server works without one.)
